Features

주요 기능

단순한 관리 봇을 넘어, AI 기반 게임 코칭까지 제공하는 올인원 디스코드 봇

🎯

AI 심층 분석 (/분석)

Match-v5 타임라인에서 사망 위치, 골드차, 오브젝트 참여를 추출하고 AI 코치가 1080px 이미지 리포트로 코칭합니다.

AI Coaching
📊

메타 적응 코칭 (/메타)

최근 50게임을 분석하여 공격성/로밍/시야 등 플레이스타일을 프로파일링하고, 패치 변경사항에 맞는 맞춤 전략을 제공합니다.

AI Coaching
⚔️

실시간 게임 분석 (/전적)

소환사 등록 후 게임 시작 시 자동 감지, 10명 참가자의 티어/챔피언을 AI가 실시간으로 분석합니다.

Game Analysis
📰

패치노트 자동 요약

30분 간격으로 새 패치를 감지하고, AI가 챔피언 버프/너프/아이템 변경을 자동 요약하여 채널에 알립니다.

AI Summary
🛠️

서버 자동 구성

게임/스터디/프로젝트/커뮤니티/비즈니스 5가지 템플릿으로 채널, 역할, 권한을 한 번에 자동 생성합니다.

Management
📈

레벨 / XP 시스템

메시지 활동 기반 경험치 지급, 레벨업 알림, 서버 순위표(TOP 10)로 커뮤니티 활성화를 돕습니다.

Gamification
👥

채널 / 역할 / 멤버 관리

채널 생성/삭제/이동/잠금, 역할 생성/부여/색상 변경, 멤버 킥/밴/타임아웃을 슬래시 명령어로 처리합니다.

Management
🌐

웹 대시보드

Discord OAuth2 로그인 후 웹 브라우저에서 서버를 관리합니다. 채널/역할/멤버/템플릿을 GUI로 제어합니다.

Dashboard
Commands

슬래시 명령어

총 15개의 슬래시 명령어로 서버의 모든 것을 제어합니다

/분석 AI 코치 심층 분석 (이미지 리포트)
/메타 50게임 플레이스타일 진단 + 패치 코칭
/전적 롤 계정 등록, 실시간/최근전적 AI 분석
/패치노트 최신 패치노트 AI 요약 + 자동 알림
/서버구성 5가지 템플릿으로 서버 자동 세팅
/채널 생성 / 삭제 / 이동 / 잠금 / 대량생성
/역할 생성 / 삭제 / 부여 / 색상변경 / 전체부여
/멤버 정보 / 킥 / 밴 / 타임아웃 / 닉네임
/레벨 내정보 / 순위 / 유저 정보
/환영 환영 메시지 설정 + 게임 역할 선택
/서버설정 서버 이름 / 인증레벨 / 알림 설정
/이벤트 서버 이벤트 생성 및 참가 관리
/멤버십 AI 크레딧 구매 + 사용 내역
/tts 텍스트를 음성으로 변환
/도움말 전체 명령어 안내
